EASY
#704
Binary Search
Search for a target value in a sorted integer array, returning its index if found.
Subtype Standard BS
Key Insight

Compare `nums[mid]` with target. If equal, return index. If target is larger, search right half (`L = mid + 1`). If target is smaller, search left half (`R = mid - 1`).

Time
O(log N)
Space
O(1)
Edge Cases

Target not in array, single-element arrays

#35
Search Insert Position
Return index of target in sorted array, or index where it would be inserted in order.
Subtype Search Position
Key Insight

Execute standard binary search. If target is not found, the left pointer `L` will end up pointing to the correct insert position when loop exits.

Time
O(log N)
Space
O(1)
Edge Cases

Insert at 0 index, insert at end of array

#69
Sqrt(x)
Calculate square root of x, rounded down to nearest integer, in constant space.
Subtype Integer Binary Search
Key Insight

Search space is 1 to x. If `mid * mid <= x`, record `mid` as a candidate solution and search right side for larger integer root. If `sq > x`, search left side.

Time
O(log X)
Space
O(1)
Edge Cases

x = 0 or 1, square root is not a perfect square

EASY
#278
First Bad Version
Find the first bad version in versions 1..n that causes all subsequent ones to fail.
Subtype Boundary Search
Key Insight

If `mid` is bad, search left half including `mid` (`R = mid`). If `mid` is good, the first bad version must be in the right half (`L = mid + 1`).

Time
O(log N)
Space
O(1)
Edge Cases

First version is bad, only the last version is bad

MEDIUM
#153
Min in Rotated Sorted Array
Find the minimum element in a sorted array that has been rotated.
Subtype Rotated Boundaries
Key Insight

Compare `nums[mid]` with `nums[R]`. If `nums[mid] > nums[R]`, pivot (minimum) is in the right half (`L = mid + 1`). Otherwise it is at or to the left (`R = mid`).

Time
O(log N)
Space
O(1)
Edge Cases

Array not rotated, array size 1 or 2

MEDIUM
#33
Search in Rotated Array
Search for a target value in a possibly rotated sorted array.
Subtype Interval Search
Key Insight

Identify which half (left/right) is sorted. If target lies within the boundaries of the sorted half, search there. Otherwise, search the other half.

Time
O(log N)
Space
O(1)
Edge Cases

Target not in array, rotation pivot at index 0

MEDIUM
#875
Koko Eating Bananas
Find the minimum integer eating speed k to eat all bananas in h hours.
Subtype Search on Range
Key Insight

Binary search on eating speed `K` between 1 and `max(piles)`. Test speed `mid`; if valid, record and look for slower speed (`R = mid - 1`).

Time
O(N * log(Max))
Space
O(1)
Edge Cases

Hours h equals piles length, piles with single elements
