#146
LRU Cache
Design a Least Recently Used (LRU) cache with O(1) average time complexity for both get and put operations.
Subtype Hash Map + Doubly Linked List
Key Insight

Map references keys to node pointers for O(1) lookups. Doubly Linked List tracks access order (MRU at head, evict LRU from tail) in O(1).

Time
O(1) average
Space
O(Capacity)
Edge Cases

Capacity of 1, update existing keys, evicting the only item in cache

MEDIUM
#380
Insert Delete GetRandom O(1)
Design a structure supporting insert, remove, and getRandom in average O(1) time.
Subtype Hash Map + Array
Key Insight

Use array for O(1) index access. Map tracks element array indices. To delete, swap target element with last element, update map index, and pop.

Time
O(1) average
Space
O(N)
Edge Cases

Removing the last element, inserting existing values
