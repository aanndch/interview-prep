EASY
#703
Kth Largest Element in a Stream
Design a class to find the kth largest element in a dynamically growing stream of numbers.
Subtype Min-Heap
Key Insight

Maintain a min-heap of size K. The root of the heap always stores the Kth largest element. Evict root if size exceeds K.

Time
O(log K) per add
Space
O(K)
Edge Cases

Stream size smaller than K, adding duplicate values

MEDIUM
#347
Top K Frequent Elements
Given an integer array, return the k most frequent elements in the array.
Subtype Min-Heap / Bucket
Key Insight

Track frequencies in a Hash Map. Keep a Min-Heap of size K. Push pairs and evict smaller frequencies to leave the top K elements.

Time
O(N log K)
Space
O(N)
Edge Cases

K equals unique count, items with identical frequencies

MEDIUM
#973
K Closest Points to Origin
Find the k closest points to the origin (0, 0) on a 2D plane.
Subtype Max-Heap
Key Insight

Maintain a max-heap of size K based on Euclidean distance. Pop the max distance node when new points are closer to origin.

Time
O(N log K)
Space
O(K)
Edge Cases

Points with identical distances, K equals points length

MEDIUM
#215
Kth Largest Element in Array
Find the kth largest element in an unsorted array without sorting it first.
Subtype Min-Heap / Select
Key Insight

Iterate through array, maintaining a Min-Heap of size K. Evict root elements that are smaller than current value. Root holds the Kth largest.

Time
O(N log K)
Space
O(K)
Edge Cases

Negative numbers, duplicates, K equals array length
