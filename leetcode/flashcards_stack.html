Valid Parentheses
Determine if a string with brackets '(' '[' '{' has correct opening and closing matches.
Subtype Balancing Stack
Key Insight

Push expected closing brackets onto stack. When closing bracket is seen, pop stack and check for mismatch or empty stack.

Time
O(N)
Space
O(N)
Edge Cases

Only open brackets, starts with closing bracket

EASY
#232
Implement Queue using Stacks
Implement a FIFO queue using only two LIFO stacks.
Subtype Dual Stacks
Key Insight

Push elements to `inStack`. For popping or peeking, if `outStack` is empty, pop all elements from `inStack` to `outStack` to reverse order.

Time
O(1) amortized
Space
O(N)
Edge Cases

Popping from empty queue, consecutive push/pop operations
