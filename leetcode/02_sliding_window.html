Fixed vs Variable Windows

Sliding Window handles contiguous subarrays. It is split into two variants:

  • Fixed-Size Window: The window width K stays constant. We initialize the sum of the first K, then slide right by subtracting the left element and adding the right element in O(1) time.
  • Variable-Size Window: The window expands by moving index R forward. If a condition is violated, index L is advanced to shrink the window until the condition is valid again.

#643 Maximum Average Subarray I

Easy
You are given an integer array nums consisting of n elements, and an integer k.

Find a contiguous subarray whose length is equal to k that has the maximum average value and return this value. Any answer with a calculation error less than 10-5 will be accepted.

This is a classic **Fixed-Size Window** problem. Instead of calculating the average of every possible subarray of size K (which takes O(N * K) time), maintain the running sum of the window.

Initialize the sum of the first K elements. For every subsequent step i, add nums[i] and subtract the element leaving the window, nums[i - K]. Find the maximum sum obtained and divide by K.

Input: nums = [1, 12, -5, -6, 50, 3], k = 4 Step 1: Compute initial window sum (i = 0..3) - Sum: 1 + 12 + (-5) + (-6) = 2 - Running Max Sum = 2 Step 2: i = 4 (value = 50) - Action: Slide window. Add nums[4] (50), Subtract nums[0] (1) - Running Sum: 2 + 50 - 1 = 51 - Max Sum: max(2, 51) = 51 Step 3: i = 5 (value = 3) - Action: Slide window. Add nums[5] (3), Subtract nums[1] (12) - Running Sum: 51 + 3 - 12 = 42 - Max Sum: max(51, 42) = 51 Result Max Average: 51 / 4 = 12.75
TypeScript 
function findMaxAverage(nums: number[], k: number): number {
    let sum = 0;
    for (let i = 0; i < k; i++) {
        sum += nums[i];
    }
    
    let maxVal = sum;
    for (let i = k; i < nums.length; i++) {
        sum += nums[i] - nums[i - k];
        maxVal = Math.max(maxVal, sum);
    }
    return maxVal / k;
}
Time Complexity: O(N)
Space Complexity: O(1)

#209 Minimum Size Subarray Sum

Easy
Given an array of positive integers nums and a positive integer target, return the minimal length of a contiguous subarray of which the sum is greater than or equal to target. If there is no such subarray, return 0 instead.

This is a **Variable-Size Window** problem. We need to find the smallest contiguous subarray whose sum is at least target.

We expand the window by moving R to the right and adding elements to a running sum. When the sum is greater than or equal to the target, we record the current window size (R - L + 1) and advance the left pointer L to shrink the window, searching for a smaller valid size.

Input: target = 7, nums = [2, 3, 1, 2, 4, 3] Step 1: R = 0 | nums[0] = 2 | runningSum = 2 - sum < target (2 < 7) -> Expand right Step 2: R = 3 | nums[3] = 2 | runningSum = 8 - sum >= target (8 >= 7) -> Record length: R - L + 1 = 4 - Shrink: Subtract nums[L](2) from left. L becomes 1. runningSum = 6. Step 3: R = 4 | nums[4] = 4 | runningSum = 10 - sum >= target (10 >= 7) -> Record length: min(4, 4) = 4 - Shrink: Subtract nums[L](3). L becomes 2. runningSum = 7 - sum >= target (7 >= 7) -> Record length: min(4, 3) = 3 (subarray: [1, 2, 4]) - Shrink: Subtract nums[L](1). L becomes 3. runningSum = 6 Step 4: R = 5 | nums[5] = 3 | runningSum = 9 - sum >= target (9 >= 7) -> Record length: min(3, 3) = 3 - Shrink: Subtract nums[L](2). L becomes 4. runningSum = 7 - sum >= target (7 >= 7) -> Record length: min(3, 2) = 2 (subarray: [4, 3]) - Shrink: Subtract nums[L](4). L becomes 5. runningSum = 3 Minimum Size Subarray Length: 2
TypeScript 
function minSubArrayLen(target: number, nums: number[]): number {
    let l = 0;
    let sum = 0;
    let minLen = Infinity;
    
    for (let r = 0; r < nums.length; r++) {
        sum += nums[r];
        while (sum >= target) {
            minLen = Math.min(minLen, r - l + 1);
            sum -= nums[l];
            l++;
        }
    }
    return minLen === Infinity ? 0 : minLen;
}
Time Complexity: O(N)
Space Complexity: O(1)

#3 Longest Substring Without Repeating Characters

Medium
Given a string s, find the length of the longest substring without repeating characters.

We expand the right pointer R, recording the index of each visited character in a Hash Map.

If we encounter a repeating character that sits inside our current window, we must move our left pointer L past the previous occurrence of this character. We update the maximum length (R - L + 1) at every step.

Input: s = "abcabcbb" Step 1: R = 0 | char: "a" - "a" not in seen. Add map entry: {'a': 0}. - Valid window: "a" (L = 0, R = 0) -> Max Len = 1 Step 2: R = 1 | char: "b" - "b" not in seen. Add map entry: {'a': 0, 'b': 1}. - Valid window: "ab" (L = 0, R = 1) -> Max Len = 2 Step 3: R = 2 | char: "c" - "c" not in seen. Add map entry: {'a': 0, 'b': 1, 'c': 2}. - Valid window: "abc" (L = 0, R = 2) -> Max Len = 3 Step 4: R = 3 | char: "a" - "a" exists in map at index 0 >= L(0). - Action: Move L past index 0 -> L becomes 1. - Update map entry: {'a': 3, 'b': 1, 'c': 2}. - Valid window: "bca" (L = 1, R = 3) -> Max Len = 3 Step 5: R = 4 | char: "b" - "b" exists in map at index 1 >= L(1). - Action: Move L past index 1 -> L becomes 2. - Update map entry: {'a': 3, 'b': 4, 'c': 2}. - Valid window: "cab" (L = 2, R = 4) -> Max Len = 3 ... (Repeats until end of string) Result: 3
TypeScript 
function lengthOfLongestSubstring(s: string): number {
    let l = 0;
    const seen = new Map<string, number>();
    let maxLen = 0;
    
    for (let r = 0; r < s.length; r++) {
        if (seen.has(s[r]) && seen.get(s[r])! >= l) {
            l = seen.get(s[r])! + 1;
        }
        seen.set(s[r], r);
        maxLen = Math.max(maxLen, r - l + 1);
    }
    return maxLen;
}
Time Complexity: O(N)
Space Complexity: O(min(M, N)) (where M is character set size)

#438 Find All Anagrams in a String

Medium
Given two strings s and p, return an array of all the start indices of p's anagrams in s. You may return the answer in any order.

This is a **Fixed-Size Window** problem of size P.length. Count the character frequencies of p in a map.

Maintain the character counts of our sliding window in s. When counts match, record the start index L. Slide by adding s[R] and removing s[R - P.length] from counts.

Input: s = "cbaebabacd", p = "abc" Target p frequency counts: {'a':1, 'b':1, 'c':1} Step 1: Initialize first window (indices 0..2) -> substring "cba" - Window frequencies: {'a': 1, 'b': 1, 'c': 1} - Action: Matches target! Add index 0 to result. Step 2: i = 3 | Slide window to indices 1..3 -> substring "bae" - Action: Add s[3]('e'), Subtract s[0]('c') - Window frequencies: {'a': 1, 'b': 1, 'e': 1} - Action: Mismatch. Step 3: i = 4 | Slide window to indices 2..4 -> substring "aeb" - Action: Add s[4]('b'), Subtract s[1]('b') - Window frequencies: {'a': 1, 'e': 1, 'b': 1} - Action: Mismatch. ... (Continues sliding window until end of s) Result start indices array: [0, 6]
TypeScript 
function findAnagrams(s: string, p: string): number[] {
    const res: number[] = [];
    if (s.length < p.length) return res;
    
    const pCount = new Array(26).fill(0);
    const sCount = new Array(26).fill(0);
    
    for (let i = 0; i < p.length; i++) {
        pCount[p.charCodeAt(i) - 97]++;
        sCount[s.charCodeAt(i) - 97]++;
    }
    
    const matches = (a1: number[], a2: number[]) => {
        for (let i = 0; i < 26; i++) {
            if (a1[i] !== a2[i]) return false;
        }
        return true;
    };
    
    if (matches(pCount, sCount)) res.push(0);
    
    for (let i = p.length; i < s.length; i++) {
        sCount[s.charCodeAt(i) - 97]++;
        sCount[s.charCodeAt(i - p.length) - 97]--;
        
        if (matches(pCount, sCount)) {
            res.push(i - p.length + 1);
        }
    }
    return res;
}
Time Complexity: O(N)
Space Complexity: O(1) (alphabet size is bounded at 26)

#567 Permutation in String

Medium
Given two strings s1 and s2, return true if s2 contains a permutation of s1, or false otherwise.

In other words, return true if one of s1's permutations is the substring of s2.

Similar to Finding Anagrams, a permutation means s2 must contain a substring of size s1.length that has the exact character counts of s1.

Maintain the sliding window of size s1.length over s2. Use frequency arrays of size 26 and return true if any window matches the character count of s1.

Input: s1 = "ab", s2 = "eidbaooo" Target s1 frequency counts: {'a': 1, 'b': 1} Step 1: Initialize first window of size 2 -> "ei" - Window counts: {'e': 1, 'i': 1} - Action: Mismatch. Step 2: i = 2 | Slide window to -> "id" - Action: Add s2[2]('d'), Subtract s2[0]('e') - Window counts: {'i': 1, 'd': 1} - Action: Mismatch. Step 3: i = 3 | Slide window to -> "db" - Action: Add s2[3]('b'), Subtract s2[1]('i') - Window counts: {'d': 1, 'b': 1} - Action: Mismatch. Step 4: i = 4 | Slide window to -> "ba" - Action: Add s2[4]('a'), Subtract s2[2]('d') - Window counts: {'b': 1, 'a': 1} - Action: Match found! Return true instantly.
TypeScript 
function checkInclusion(s1: string, s2: string): boolean {
    if (s1.length > s2.length) return false;
    
    const count1 = new Array(26).fill(0);
    const count2 = new Array(26).fill(0);
    
    for (let i = 0; i < s1.length; i++) {
        count1[s1.charCodeAt(i) - 97]++;
        count2[s2.charCodeAt(i) - 97]++;
    }
    
    const matches = (c1: number[], c2: number[]) => {
        for (let i = 0; i < 26; i++) {
            if (c1[i] !== c2[i]) return false;
        }
        return true;
    };
    
    if (matches(count1, count2)) return true;
    
    for (let i = s1.length; i < s2.length; i++) {
        count2[s2.charCodeAt(i) - 97]++;
        count2[s2.charCodeAt(i - s1.length) - 97]--;
        
        if (matches(count1, count2)) return true;
    }
    return false;
}
Time Complexity: O(N)
Space Complexity: O(1)
