What is Sliding Window?

Sliding Window is a technique that maintains a contiguous subarray/substring (the "window") and slides it across the data structure to solve problems in O(n) time instead of O(n² or n³).

Instead of checking every possible subarray (O(n²)), we expand/shrink the window as we go:

// Before (brute force):
for i in range(n):
    for j in range(i, n):  // O(n²) subarrays

// After (sliding window):
left = 0
for right in range(n):
    // expand window
    while invalid_condition:
        // shrink from left
        left += 1

The window is defined by two pointers (left and right) that both move forward only — unlike Two Pointers where they move toward each other.

Fixed vs Variable Window

A) Fixed-Size Window

Window size = 4:

3

Both L and R move together by 1 each step. Window size stays constant.

def fixed_window(arr, k):
    # build initial window [0, k-1]
    window = sum(arr[:k])
    result = window
    for i in range(k, len(arr)):
        window += arr[i] - arr[i - k]  // slide right
        result = max(result, window)
    return result

B) Variable-Size Window

Window grows/shrinks:
aL
b
c
dR
e
f

R expands the window on each iteration. L shrinks when the condition is violated. Window size varies.

def variable_window(arr):
    l = 0
    for r in range(len(arr)):
        // expand: add arr[r] to window state
        while invalid(l, r):
            // shrink: remove arr[l] from window state
            l += 1
        // update result with valid window

Template variable-size (substring):

def sliding_window(s):
    l = 0
    counts = {}
    result = 0
    for r in range(len(s)):
        counts[s[r]] = counts.get(s[r], 0) + 1
        while shrink_condition(counts):
            counts[s[l]] -= 1
            if counts[s[l]] == 0:
                del counts[s[l]]
            l += 1
        result = max(result, r - l + 1)
    return result

Easy Problems

EASY #643 Maximum Average Subarray I

You are given an integer array nums consisting of n elements, and an integer k. Find a contiguous subarray whose length is equal to k that has the maximum average value and return this value. Any answer with a calculation error less than 10-5 will be accepted.

Approach & Walkthrough

Approach: Fixed-size sliding window. Compute sum of first k, then slide subtracting nums[i-k] and adding nums[i].

Solution Code 
def findMaxAverage(nums: list[int], k: int) -> float:
    window = sum(nums[:k])
    best = window
    for i in range(k, len(nums)):
        window += nums[i] - nums[i - k]
        best = max(best, window)
    return best / k

Time: O(n)  |  Space: O(1)

Given an array of positive integers nums and a positive integer target, return the minimal length of a subarray whose sum is greater than or equal to target. If there is no such subarray, return 0 instead.

Solution Code 
def minSubArrayLen(target: int, nums: list[int]) -> int:
    l = 0
    window = 0
    best = len(nums) + 1
    for r in range(len(nums)):
        window += nums[r]
        while window >= target:
            best = min(best, r - l + 1)
            window -= nums[l]
            l += 1
    return best if best <= len(nums) else 0

Time: O(n)  |  Space: O(1)

Given a string s, find the length of the longest substring without repeating characters.

Solution Code 
def lengthOfLongestSubstring(s: str) -> int:
    l = 0
    seen = {}
    best = 0
    for r in range(len(s)):
        if s[r] in seen and seen[s[r]] >= l:
            l = seen[s[r]] + 1
        seen[s[r]] = r
        best = max(best, r - l + 1)
    return best

Time: O(n)  |  Space: O(min(m, n)) where m = charset size

MEDIUM #438 Find All Anagrams in a String

Given two strings s and p, return an array of all the start indices of p's anagrams in s. You may return the answer in any order.

Solution Code 
def findAnagrams(s: str, p: str) -> list[int]:
    ns, np = len(s), len(p)
    if np > ns: return []
    p_count = [0] * 26
    s_count = [0] * 26
    for ch in p:
        p_count[ord(ch) - ord('a')] += 1
    res = []
    for i in range(ns):
        s_count[ord(s[i]) - ord('a')] += 1
        if i >= np:
            s_count[ord(s[i - np]) - ord('a')] -= 1
        if s_count == p_count:
            res.append(i - np + 1)
    return res

Time: O(n)  |  Space: O(1) (fixed 26-size arrays)

MEDIUM #567 Permutation in String

Given two strings s1 and s2, return true if s2 contains a permutation of s1, or false otherwise. In other words, return true if one of s1's permutations is the substring of s2.

Approach & Walkthrough

Approach: Same fixed-size window as #438. Return true as soon as window frequencies match.

Solution Code 
def checkInclusion(s1: str, s2: str) -> bool:
    n1, n2 = len(s1), len(s2)
    if n1 > n2: return False
    count1 = [0] * 26
    count2 = [0] * 26
    for ch in s1:
        count1[ord(ch) - ord('a')] += 1
    for i in range(n2):
        count2[ord(s2[i]) - ord('a')] += 1
        if i >= n1:
            count2[ord(s2[i - n1]) - ord('a')] -= 1
        if count1 == count2:
            return True
    return False

Time: O(n)  |  Space: O(1)

MEDIUM #76 Minimum Window Substring

Given two strings s and t of lengths m and n respectively, return the minimum window substring of s such that every character in t (including duplicates) is included in the window. import collections

def minWindow(s: str, t: str) -> str:
    need = collections.Counter(t)
    have = collections.defaultdict(int)
    l = 0
    formed = 0
    min_len = float("inf")
    start = 0
    for r in range(len(s)):
        ch = s[r]
        have[ch] += 1
        if ch in need and have[ch] == need[ch]:
            formed += 1
        while formed == len(need):
            if r - l + 1 < min_len:
                min_len = r - l + 1
                start = l
            have[s[l]] -= 1
            if s[l] in need and have[s[l]] < need[s[l]]:
                formed -= 1
            l += 1
    return s[start:start + min_len] if min_len != float("inf") else ""

Time: O(n)  |  Space: O(m) where m = unique chars in t

MEDIUM #239 Sliding Window Maximum

You are given an array of integers nums, there is a sliding window of size k which is moving from the very left of the array to the very right. You can only see the k numbers in the window. Each time the sliding window moves right by one position. Return the max sliding window.

import collections

def maxSlidingWindow(nums: list[int], k: int) -> list[int]:
    dq = collections.deque()
    res = []
    for i in range(len(nums)):
        # remove out-of-window indices from front
        if dq and dq[0] < i - k + 1:
            dq.popleft()
        # maintain decreasing order: remove smaller values from back
        while dq and nums[dq[-1]] < nums[i]:
            dq.pop()
        dq.append(i)
        if i >= k - 1:
            res.append(nums[dq[0]])
    return res

Time: O(n)  |  Space: O(k) for the deque

You are given a string s and an integer k. You can choose any character of the string and change it to any other uppercase English character. You can perform this operation at most k times. Return the length of the longest substring containing the same letter you can get after performing the above operations.

Solution Code 
def characterReplacement(s: str, k: int) -> int:
    freq = {}
    l = 0
    max_freq = 0
    best = 0
    for r in range(len(s)):
        freq[s[r]] = freq.get(s[r], 0) + 1
        max_freq = max(max_freq, freq[s[r]])
        while (r - l + 1) - max_freq > k:
            freq[s[l]] -= 1
            l += 1
        best = max(best, r - l + 1)
    return best

Time: O(n)  |  Space: O(m) where m = unique chars

Solution Code 
def longestOnes(nums: list[int], k: int) -> int:
    l = 0
    zeros = 0
    best = 0
    for r in range(len(nums)):
        if nums[r] == 0:
            zeros += 1
        while zeros > k:
            if nums[l] == 0:
                zeros -= 1
            l += 1
        best = max(best, r - l + 1)
    return best

Time: O(n)  |  Space: O(1)
