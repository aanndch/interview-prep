EASY
#136
Single Number
Given a non-empty array of integers where every element appears twice except for one, find that single one.
Subtype Bitwise XOR
Key Insight

XORing any number with itself results in 0 (`A ^ A = 0`). Accumulating XOR sum across all array elements cancels pairs, leaving the single target.

Time
O(N)
Space
O(1)
Edge Cases

Negative values, single-element arrays

EASY
#191
Number of 1 Bits
Write a function that takes an unsigned integer and returns the number of '1' bits it has (also known as the Hamming weight).
Subtype Bit Mask / Bit Shifting
Key Insight

Use `n = n & (n - 1)` which deletes the lowest set bit in constant O(1) steps. Repeat until n becomes 0, incrementing the count.

Time
O(1) (max 32 steps)
Space
O(1)
Edge Cases

Input is 0, maximum 32-bit unsigned integer value
