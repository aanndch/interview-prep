Pointer Manipulation Strategies

Linked list problems check your ability to change object pointers without losing references. Key techniques include:

  • Dummy Head: Always create a dummy pre-head node (e.g. `dummy = new ListNode(-1)`) to handle edge cases easily, such as deleting the head node or returning a new list.
  • Two Pointers (Slow/Fast): Crucial for finding list midpoints, detecting cycles, or obtaining the Nth node from the end.

#206 Reverse Linked List

Easy
Given the head of a singly linked list, reverse the list, and return the reversed list.

Maintain three pointers: prev (initially null), curr (initially head), and next (temporary pointer).

Iterate through the list. Point curr.next backward to prev. Then shift prev to curr, and move curr to the next node (stored in next).

Input: head = [1, 2, 3] Step 1: curr = node(1), prev = null - next = curr.next = node(2) - curr.next = prev = null - prev = curr = node(1) - curr = next = node(2) Step 2: curr = node(2), prev = node(1) - next = curr.next = node(3) - curr.next = prev = node(1) - prev = curr = node(2) - curr = next = node(3) Step 3: curr = node(3), prev = node(2) - next = curr.next = null - curr.next = prev = node(2) - prev = curr = node(3) - curr = next = null Result: prev points to node(3). Reversed list: 3 -> 2 -> 1 -> null
TypeScript 
class ListNode {
    val: number;
    next: ListNode | null;
    constructor(val?: number, next?: ListNode | null) {
        this.val = (val===undefined ? 0 : val);
        this.next = (next===undefined ? null : next);
    }
}

function reverseList(head: ListNode | null): ListNode | null {
    let prev: ListNode | null = null;
    let curr = head;
    
    while (curr !== null) {
        const nextNode = curr.next;
        curr.next = prev;
        prev = curr;
        curr = nextNode;
    }
    return prev;
}
Time Complexity: O(N)
Space Complexity: O(1)

#141 Linked List Cycle

Easy
Given head, the head of a linked list, determine if the linked list has a cycle in it.

Use Floyd's Cycle-Finding Algorithm (Tortoise and Hare). Maintain two pointers: slow and fast.

Advance slow by 1 step and fast by 2 steps. If there is no cycle, fast will reach null. If there is a cycle, fast will eventually lap and meet slow.

Input: head = [3, 2, 0, -4], cycle at index 1 Step 1: slow = node(3), fast = node(3) Step 2: slow = slow.next = node(2) - fast = fast.next.next = node(0) - slow !== fast Step 3: slow = slow.next = node(0) - fast = fast.next.next = node(2) - slow !== fast Step 4: slow = slow.next = node(-4) - fast = fast.next.next = node(-4) - slow === fast -> Meeting detected! Cycle exists. Result: true
TypeScript 
function hasCycle(head: ListNode | null): boolean {
    let slow = head;
    let fast = head;
    
    while (fast !== null && fast.next !== null) {
        slow = slow!.next;
        fast = fast.next.next;
        if (slow === fast) {
            return true;
        }
    }
    return false;
}
Time Complexity: O(N)
Space Complexity: O(1)

#21 Merge Two Sorted Lists

Easy
You are given the heads of two sorted linked lists list1 and list2. Merge the two lists into one sorted list. The list should be made by splicing together the nodes of the first two lists.

Use a dummy pre-head node. Compare head values of list1 and list2. Link the smaller node to the merged list and advance its pointer. Repeat until one list is empty, then append the remaining list directly.

Input: list1 = [1, 2, 4], list2 = [1, 3, 4] Dummy = ListNode(-1), curr = Dummy Step 1: list1(1) === list2(1) -> Link list1(1). Advance list1. curr = node(1) Step 2: list1(2) > list2(1) -> Link list2(1). Advance list2. curr = node(1) Step 3: list1(2) < list2(3) -> Link list1(2). Advance list1. curr = node(2) Step 4: list1(4) > list2(3) -> Link list2(3). Advance list2. curr = node(3) Step 5: list1(4) === list2(4) -> Link list1(4). Advance list1. curr = node(4) Step 6: list1 is null. Append remaining list2(4) node. curr.next = list2 Result: Dummy.next = 1 -> 1 -> 2 -> 3 -> 4 -> 4
TypeScript 
function mergeTwoLists(list1: ListNode | null, list2: ListNode | null): ListNode | null {
    const dummy = new ListNode(-1);
    let curr = dummy;
    
    while (list1 !== null && list2 !== null) {
        if (list1.val <= list2.val) {
            curr.next = list1;
            list1 = list1.next;
        } else {
            curr.next = list2;
            list2 = list2.next;
        }
        curr = curr.next;
    }
    
    curr.next = list1 !== null ? list1 : list2;
    return dummy.next;
}
Time Complexity: O(N + M)
Space Complexity: O(1)

#160 Intersection of Two Linked Lists

Easy
Given the heads of two singly linked-lists headA and headB, return the node at which the two lists intersect. If the two linked lists have no intersection at all, return null.

Walk pointers `A` and `B` simultaneously. When a pointer reaches the end, reset it to the head of the *other* list.

Since this offsets the length difference between the two lists, the pointers will align on their second pass, meeting exactly at the intersection node (or null if no intersection exists).

Input: List A = [4, 1, 8, 4, 5], List B = [5, 6, 1, 8, 4, 5] Intersection starts at node 8. Length(A)=5, Length(B)=6. Pass 1: - ptrA starts at A(4), ptrB starts at B(5). - ptrA reaches end after 5 steps. Reset ptrA to headB(5). - ptrB reaches end after 6 steps. Reset ptrB to headA(4). Pass 2: - Walk pointers. They align. - They meet at node(8) after 2 more steps. Result: node(8)
TypeScript 
function getIntersectionNode(headA: ListNode | null, headB: ListNode | null): ListNode | null {
    if (headA === null || headB === null) return null;
    let a = headA;
    let b = headB;
    
    while (a !== b) {
        a = a === null ? headB : a.next!;
        b = b === null ? headA : b.next!;
    }
    return a;
}
Time Complexity: O(N + M)
Space Complexity: O(1)

#83 Remove Duplicates from Sorted List

Easy
Given the head of a sorted linked list, delete all duplicates such that each element appears only once. Return the linked list sorted as well.

Since the list is sorted, duplicate elements will be adjacent. Traverse the list. If curr.val === curr.next.val, delete the duplicate by pointing curr.next = curr.next.next. Otherwise, advance `curr`.

Input: head = [1, 1, 2] Step 1: curr = node(1), next = node(1) - values match (1 === 1) -> curr.next = curr.next.next = node(2). List: 1 -> 2 Step 2: curr = node(1), next = node(2) - mismatch (1 !== 2) -> curr = curr.next = node(2) Step 3: curr.next is null -> Loop terminates. Result: 1 -> 2 -> null
TypeScript 
function deleteDuplicates(head: ListNode | null): ListNode | null {
    let curr = head;
    while (curr !== null && curr.next !== null) {
        if (curr.val === curr.next.val) {
            curr.next = curr.next.next;
        } else {
            curr = curr.next;
        }
    }
    return head;
}
Time Complexity: O(N)
Space Complexity: O(1)

#234 Palindrome Linked List

Easy
Given the head of a singly linked list, return true if it is a palindrome or false otherwise.

To solve in O(N) time and O(1) space:

1. Find midpoint of the list using slow/fast pointers.
2. Reverse the second half of the list starting from the midpoint.
3. Compare the first half and reversed second half node-by-node. If mismatch occurs, return false.

Input: head = [1, 2, 2, 1] Step 1: Find midpoint. slow stops at index 2 (node(2)). Step 2: Reverse second half. Reversed half starts at 1: 1 -> 2 -> null Step 3: Compare head [1, 2] and reversed half [1, 2]: - head(1) === reversed(1) -> Matches - head(2) === reversed(2) -> Matches Result: true
TypeScript 
function isPalindrome(head: ListNode | null): boolean {
    if (head === null || head.next === null) return true;
    
    let slow: ListNode | null = head;
    let fast = head;
    while (fast !== null && fast.next !== null) {
        slow = slow!.next;
        fast = fast.next.next;
    }
    
    let prev: ListNode | null = null;
    let curr = slow;
    while (curr !== null) {
        const nextNode = curr.next;
        curr.next = prev;
        prev = curr;
        curr = nextNode;
    }
    
    let p1: ListNode | null = head;
    let p2 = prev;
    while (p2 !== null) {
        if (p1!.val !== p2.val) {
            return false;
        }
        p1 = p1!.next;
        p2 = p2.next;
    }
    return true;
}
Time Complexity: O(N)
Space Complexity: O(1)

#19 Remove Nth Node From End of List

Medium
Given the head of a linked list, remove the nth node from the end of the list and return its head.

Use two pointers fast and slow. Advance fast by n steps first. This creates a gap of size N between them.

Walk both pointers together. When fast.next reaches null, slow will point to the node immediately preceding the node to delete. Re-link slow.next = slow.next.next.

Input: head = [1, 2, 3, 4, 5], n = 2 Dummy = node(-1), dummy.next = head fast = Dummy, slow = Dummy Step 1: Advance fast by 2 steps -> fast points to node(2). Step 2: Walk both fast and slow until fast reaches the end node(5): - slow stops at node(3). Step 3: Delete target: slow.next = slow.next.next (node(3).next = node(5)). Result: 1 -> 2 -> 3 -> 5 -> null
TypeScript 
function removeNthFromEnd(head: ListNode | null, n: number): ListNode | null {
    const dummy = new ListNode(-1, head);
    let fast: ListNode | null = dummy;
    let slow: ListNode | null = dummy;
    
    for (let i = 0; i < n; i++) {
        fast = fast!.next;
    }
    
    while (fast!.next !== null) {
        fast = fast!.next;
        slow = slow!.next;
    }
    
    slow!.next = slow!.next!.next;
    return dummy.next;
}
Time Complexity: O(N)
Space Complexity: O(1)

#2 Add Two Numbers

Medium
You are given two non-empty linked lists representing two non-negative integers. The digits are stored in reverse order, and each of their nodes contains a single digit. Add the two numbers and return the sum as a linked list.

Walk both lists node-by-node. Add node values along with a running carry.

Create a new node with value `sum % 10` and update `carry = Math.floor(sum / 10)`. Continue until both lists are empty and no carry remains.

Input: l1 = [2, 4, 3], l2 = [5, 6, 4] Step 1: val1 = 2, val2 = 5, carry = 0 -> sum = 7. New node(7). carry = 0 Step 2: val1 = 4, val2 = 6, carry = 0 -> sum = 10. New node(0). carry = 1 Step 3: val1 = 3, val2 = 4, carry = 1 -> sum = 8. New node(8). carry = 0 Result: 7 -> 0 -> 8 -> null
TypeScript 
function addTwoNumbers(l1: ListNode | null, l2: ListNode | null): ListNode | null {
    const dummy = new ListNode(-1);
    let curr = dummy;
    let carry = 0;
    
    while (l1 !== null || l2 !== null || carry !== 0) {
        const val1 = l1 !== null ? l1.val : 0;
        const val2 = l2 !== null ? l2.val : 0;
        const sum = val1 + val2 + carry;
        
        carry = Math.floor(sum / 10);
        curr.next = new ListNode(sum % 10);
        curr = curr.next;
        
        if (l1 !== null) l1 = l1.next;
        if (l2 !== null) l2 = l2.next;
    }
    return dummy.next;
}
Time Complexity: O(max(N, M))
Space Complexity: O(max(N, M))

#142 Linked List Cycle II

Medium
Given the head of a linked list, return the node where the cycle begins. If there is no cycle, return null.

Use Floyd's Cycle Detection. If slow and fast pointers meet, a cycle exists.

Reset one pointer to head. Walk both pointers 1 step at a time. The point where they meet is the start node of the cycle.

Input: head = [3, 2, 0, -4], cycle at index 1 Step 1: Meeting point found at node(-4). Step 2: Reset slow pointer to head node(3). Step 3: Walk both slow and fast 1 step at a time: - slow goes from node(3) to node(2). - fast goes from node(-4) to node(2). - They meet at node(2). Result: Node(2)
TypeScript 
function detectCycle(head: ListNode | null): ListNode | null {
    let slow = head;
    let fast = head;
    
    while (fast !== null && fast.next !== null) {
        slow = slow!.next;
        fast = fast.next.next;
        if (slow === fast) {
            let entry = head;
            while (entry !== slow) {
                entry = entry!.next;
                slow = slow!.next;
            }
            return entry;
        }
    }
    return null;
}
Time Complexity: O(N)
Space Complexity: O(1)

#24 Swap Nodes in Pairs

Medium
Given a singly linked list, swap every two adjacent nodes and return its head. You must solve the problem without modifying the values in the list's nodes (i.e., only nodes themselves may be changed.)

Use a dummy node pointing to head. Maintain a prev pointer starting at dummy.

For each pair of nodes (first and second), re-link: prev.next = second, first.next = second.next, and second.next = first. Then advance prev = first for the next pair.

Input: head = [1, 2, 3, 4] Dummy = node(-1), dummy.next = head, prev = Dummy Step 1: first = node(1), second = node(2) - prev.next = second (dummy -> 2) - first.next = second.next (1 -> 3) - second.next = first (2 -> 1) - prev = first = node(1). List: dummy -> 2 -> 1 -> 3 -> 4 Step 2: first = node(3), second = node(4) - prev.next = second (1 -> 4) - first.next = second.next (3 -> null) - second.next = first (4 -> 3) - prev = first = node(3). List: dummy -> 2 -> 1 -> 4 -> 3 -> null Result: 2 -> 1 -> 4 -> 3 -> null
TypeScript 
function swapPairs(head: ListNode | null): ListNode | null {
    const dummy = new ListNode(-1, head);
    let prev = dummy;
    
    while (prev.next !== null && prev.next.next !== null) {
        const first = prev.next;
        const second = prev.next.next;
        
        first.next = second.next;
        second.next = first;
        prev.next = second;
        
        prev = first;
    }
    return dummy.next;
}
Time Complexity: O(N)
Space Complexity: O(1)

#328 Odd Even Linked List

Medium
Given the head of a singly linked list, group all the nodes with odd indices together followed by the nodes with even indices, and return the reordered list.

Maintain two separate lists: odd (starting at head) and even (starting at head.next).

Iterate linking odd.next = odd.next.next and even.next = even.next.next to skip alternate nodes. Finally, link the end of the odd list to the head of the even list.

Input: head = [1, 2, 3, 4, 5] odd = node(1), even = node(2), evenHead = node(2) Step 1: odd.next = node(3). odd = node(3) - even.next = node(4). even = node(4) Step 2: odd.next = node(5). odd = node(5) - even.next = null. even = null Step 3: Link lists: odd.next = evenHead (5 -> 2) Result: 1 -> 3 -> 5 -> 2 -> 4 -> null
TypeScript 
function oddEvenList(head: ListNode | null): ListNode | null {
    if (head === null) return null;
    let odd = head;
    let even = head.next;
    const evenHead = even;
    
    while (even !== null && even.next !== null) {
        odd.next = even.next;
        odd = odd.next;
        even.next = odd.next;
        even = even.next;
    }
    odd.next = evenHead;
    return head;
}
Time Complexity: O(N)
Space Complexity: O(1)
