EASY
#206
Reverse Linked List
Reverse a singly linked list in-place and return the new head.
Subtype Pointer Reversal
Key Insight

Maintain `prev` and `curr`. Point `curr.next` back to `prev`. Advance pointers `prev = curr` and `curr = nextNode` in sequence.

Time
O(N)
Space
O(1)
Edge Cases

Empty list, list with single node

EASY
#141
Linked List Cycle
Determine if a singly linked list contains a cycle.
Subtype Two Pointers
Key Insight

Use Floyd's tortoise/hare. Move `slow` by 1 node, `fast` by 2 nodes. If pointers meet, list has cycle. If `fast` hits null, no cycle.

Time
O(N)
Space
O(1)
Edge Cases

No cycle, size 2 list with cycle

#21
Merge Two Sorted Lists
Merge two sorted linked lists into one sorted linked list.
Subtype Dummy Node
Key Insight

Use dummy pre-head node. Compare head values of lists; attach smaller to `curr.next` and advance. Link remaining nodes at end.

Time
O(N + M)
Space
O(1)
Edge Cases

One or both lists empty

EASY
#160
Intersection of Two Lists
Find node where two singly linked lists intersect. Return null if none.
Subtype Cycle Sync Pointers
Key Insight

Walk ptrA and ptrB. When either hits null, redirect to other list's head. They align on second pass and meet at intersection.

Time
O(N + M)
Space
O(1)
Edge Cases

No intersection, lists of identical length

EASY
#83
Remove Duplicates from Sorted
Delete duplicate nodes from sorted linked list so elements are unique.
Subtype Adjacent Deduplication
Key Insight

Compare `curr.val` with `curr.next.val`. If identical, skip adjacent duplicate node by re-linking: `curr.next = curr.next.next`.

Time
O(N)
Space
O(1)
Edge Cases

All duplicate elements, no duplicates

Palindrome Linked List
Check if a singly linked list is a palindrome.
Subtype Reverse Second Half
Key Insight

Find list mid using slow/fast. Reverse the second half of the list. Compare first half and reversed second half node-by-node.

Time
O(N)
Space
O(1)
Edge Cases

Odd vs even list lengths, single-element list

#19
Remove Nth Node from End
Remove the nth node from the end of list and return its head.
Subtype Gap Sync Pointers
Key Insight

Advance `fast` by `n` nodes. Walk both slow and fast together until `fast.next` is null. Delete target node `slow.next`.

Time
O(N)
Space
O(1)
Edge Cases

Removing the head node, list of size 1

#2
Add Two Numbers
Add two numbers represented by linked lists in reverse order.
Subtype Sum Carry Node
Key Insight

Walk both lists. Add digits together with carry. Create new node for `sum % 10` and save `carry = sum / 10`.

Time
O(max(N, M))
Space
O(max(N, M))
Edge Cases

Final digit sum has carry, different length lists

#142
Linked List Cycle II
Detect if a cycle exists, returning the exact node where cycle begins.
Subtype Cycle Meeting Point
Key Insight

Find tortoise/hare meeting node. Reset one pointer to head. Walk both pointers 1 step at a time; they meet at cycle entry.

Time
O(N)
Space
O(1)
Edge Cases

No cycle in list, cycle loops to head node

MEDIUM
#24
Swap Nodes in Pairs
Swap every two adjacent nodes in singly linked list in-place.
Subtype Adjacent Swap
Key Insight

Use dummy. For nodes `first`/`second` swap: `prev.next = second`, `first.next = second.next`, `second.next = first`. Shift `prev = first`.

Time
O(N)
Space
O(1)
Edge Cases

Odd list size (last node remains), empty list

MEDIUM
#328
Odd Even Linked List
Group all odd-indexed nodes together followed by even-indexed nodes.
Subtype Partition Sync
Key Insight

Create `odd` and `even` pointers. Traverse linking odd-to-odd and even-to-even. Attach even head to odd list end.

Time
O(N)
Space
O(1)
Edge Cases

List length smaller than 3, even/odd total node counts
