Implement Trie (Prefix Tree)
Implement a trie with insert, search, and startsWith methods.
Subtype Trie Node Design
Key Insight

Use nodes containing a mapping of child characters to next TrieNodes and a boolean flag indicating the end of a word.

Time
O(L) per word operation
Space
O(Alphabet * WordCount)
Edge Cases

Empty prefixes, searches for words that are sub-words of stored keys

Design Add/Search Words
Design a data structure that supports adding new words and searching with wildcard '.' support.
Subtype Trie + DFS Backtracking
Key Insight

Standard Trie insertion. For wildcard search queries ('.'), use DFS recursion to explore all child paths at that character position.

Time
O(Alphabet^L) worst search
Space
O(Alphabet * WordCount)
Edge Cases

Queries with multiple wildcards, queries with wildcards at boundaries
